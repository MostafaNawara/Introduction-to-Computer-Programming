## JAVA Data Types
------------------

- #### String
 	- Mostafa
	- Egypt
	- me@nawara.me
- #### int
 	- 1
	- 500
	- -65
	- 125
- #### Float
	- 1.5
	- -99.8
	- 0.2
	- 5.0
- #### Long
	- Like the **int** but it's size is 64bit, when the int is only 32bit
- #### Double
	- Like the **Float** but it's size is 64bit, when the Float is only 32bit
